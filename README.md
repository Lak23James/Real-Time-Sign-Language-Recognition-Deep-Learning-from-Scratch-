    # 🤟 Real-Time Sign Language Recognition

A deep learning project built from scratch to recognize American Sign Language (ASL) alphabets in real time. This repository contains the complete pipeline, from training a custom neural network on the Sign Language MNIST dataset to running real-time webcam inference.

## 📂 Project Structure

* **`Train.ipynb`**: The core Jupyter Notebook containing data preprocessing, model architecture, and the training loop.
* **`test_brain.py`**: The main execution script for real-time inference. It captures live video feed and uses the trained model to predict sign language gestures on the fly.
* **`sign_mnist_test.csv`**: The testing dataset, formatted similarly to the classic MNIST dataset, containing flattened pixel values for hand gesture images.
* **`american_sign_language.PNG` / `amer_sign2.png` / `amer_sign3.png`**: Reference diagrams illustrating the ASL alphabet mappings used for the model's classification.
